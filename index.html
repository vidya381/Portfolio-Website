Featured Projects

NumVana

       

NumVana is an engaging web-based number guessing game that combines strategy, intuition, and quick thinking. Featuring three difficulty levels - Easy, Medium, and Hard - it challenges players to deduce a hidden number within a time limit. Built with HTML5, CSS3, and JavaScript, NumVana boasts a responsive design that ensures seamless gameplay across all devices. The game's robust backend, powered by Spring Boot, manages the core logic through RESTful APIs, providing a smooth and interactive user experience. Players receive instant feedback on their guesses, allowing them to refine their strategy in real-time. With dynamic features like score tracking, an attempt counter, and an interactive timer, NumVana keeps players engaged and coming back for more. Whether you're a casual gamer or a number enthusiast, NumVana transforms numerical challenges into an addictive and brain-teasing adventure. Test your numerical prowess and strategic thinking at NumVana today!

Blog Alchemy

     

BlogAlchemy is a feature-rich web-based blogging platform built with Spring Boot. It offers user authentication, role-based access control, and intuitive post management. Key features include post scheduling, image uploads, and a nested comment system. The project uses Thymeleaf for server-side rendering and Spring Security for authentication, demonstrating scalable web application development. BlogAlchemy showcases advanced backend techniques with clean architecture, modular design, and efficient database management using JPA and Hibernate.
